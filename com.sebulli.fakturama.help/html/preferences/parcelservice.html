Paketdienst

Auswahl des Paketdienstes aus einer Liste. Wird der Eintrag Eigene Einstellungen verwenden selektiert, so werden die Werte der nachfolgenden Felder verwendet. Ansonsten werden bereits im Programm hinterlegte Werte für den jeweiligen Paketdienst benutzt.

URL des Paketdienstes

URL der Webseite, die zum Ausfüllen eines Online-Paketscheines dient. Es ist auch möglich, eine Seite vor derjenigen mit den Formularfeldern anzugeben.

Absender ...

Name der Formularfelder der Absenderadresse.

Empfänger ...

Name der Formularfelder der Empfängeradresse.

Verwenden von eigenen Einstellungen

Soll ein Paketdienst verwendet werden, der nicht bereits in der Auswahlliste aufgeführt ist, so können für diesen die URL und die Namen der Formularfelder angegeben werden.

In den Feldern für Absender und Empfänger sind die Namen der HTML Formularfelder anzugeben. Diese werden dann entsprechend mit Inhalt gefüllt. Als Absenderadresse wird diejenige verwendet, die unter Einstellungen Firmendaten angegeben wurde.
